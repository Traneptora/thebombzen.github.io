Tanks

Because Oracle now requires that a trusted source sign all Java applets and WebStart applications (which costs hundreds of dollars), I will only provide this program as a downloadable jarfile.
Download Tanks

This is a preliminary version (dev31), so it will probably contain bugs.

Tanks uses a random fractal method adapted from Hugo Elias's perlin noise page to generate the terrain and a four-dimensional Runge-Kutta method to track the moving objects (the four dimensions are the x and y components of both the position and velocity). Since version dev27 it relies heavily on vector arithmetic and not on component arithmetic, and has code for delta-forced projectiles, terrain surface normal vectors, vector reflections, inverse-square-law-decaying forces, and it all sits on standard kinematics.

Tanks is in the public domain. You can do whatever you want with it as long as you don't break any laws in your jurisdiction.
